## 게시판 api 구현 프로젝트
### ✅ 학습 목표

- URL, Http Method를 활용한 기본 라우팅 방법 학습
- Request Body, Query Params, Path Params 사용해보기
- ORM 사용
    - 엔티티(테이블) 생성
    - 기본적인 CRUD 활용
- 요청부터 응답까지 전체 과정 경험
- REST API 설계
- HTTP Status Code (상태 코드)
- Postman 사용법

### ✅ 요구 사항

1. **게시글 작성 기능**

1. **게시글 전체 조회 기능**

1. **특정 게시글 조회 기능**

1. **특정 게시글 수정 기능**

1. **특정 게시글 삭제 기능**
